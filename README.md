## 📖 Overview

**EV Fleet Intelligence Suite** is an AI-powered platform for monitoring
EV battery health, predicting maintenance needs, evaluating fleet
electrification readiness, and estimating sustainability impact through
interactive dashboards.

## ✨ Features

-   🔋 Battery SOH Prediction
-   ⏳ Remaining Useful Life (RUL)
-   🛠 Predictive Maintenance
-   🚗 Fleet Electrification Readiness
-   🌱 Carbon Intelligence Tracker
-   📊 Interactive Dashboard

## 🧠 Machine Learning

Supports: - SOH Prediction - RUL Prediction - Thermal Risk Assessment -
Health Classification - Service Recommendation
